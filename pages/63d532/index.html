# markdown快速上手

# 简介

介绍

Markdown 是一种轻量级标记语言。它为网页写作者提供了一种简单的文本与html转换手段。

作者John Gruber的描述：

Markdown is a text-to-HTML conversion tool for web writers. Markdown allows you to write using an easy-to-read, easy-to-write plain text format, then convert it to structurally valid XHTML (or HTML).

# 使用markdown创作文章

  • 可以选择使用专门的markdown编辑器进行编辑，typora (opens new window)是一款比较知名的付费编辑器，但使用vscode (opens new window)配合插件也能有不错的体验

  • markdown文件以.md为后缀名

  • markdown的一大特点在于：不需要像word一样管理文章、字体等内容的格式，只需关注文章整体的结构与内容的重点。

  • 写作markdown文档时，可以通过各级标题将文章的要点进行分割，使得文章更清晰明了。

  • 在写作markdown文档时也可以使用html语法。部分情况下（如表格较大）可以加快渲染速度。

  • 利用markdown容器 (opens new window)可以丰富页面的效果
